# dva-demo

## Usage

Install dependencies

```bash
$ npm install
```

Use json server for mock data

```bash
$ npm install -g json-server
$ cd ./src/data
$ json-server --watch db.json
```

Start server
```bash
$ npm start
```

## Detail

计数器：http://localhost:8000/#/count  
用户管理：http://localhost:8000/#/authors  
课程管理：http://localhost:8000/#/courses
